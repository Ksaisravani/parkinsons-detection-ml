# Parkinson’s Disease Detection using Deep Learning

## 📌 Overview
This project detects Parkinson’s disease using handwriting images with deep transfer learning techniques.

## 🚀 Features
- CNN models (VGG19, ResNet50, InceptionV3)
- KNN classifier with optimized parameters
- Flask web application
- User login & real-time prediction

## 🛠️ Tech Stack
- Python
- TensorFlow / Keras
- Flask
- HTML, CSS, JavaScript
- SQLite

## 📊 Result
Achieved better accuracy compared to traditional SVM methods.
